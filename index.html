Projetos

Confeitaria Eliana

Criei o site da Confeitaria Eliana utilizando WordPress e o Elementor. Este projeto destaca a experiência da minha mãe, Eliana, na transformação de bolos em obras-primas, com um foco especial em bolos temáticos e de aniversário. O site foi desenvolvido para refletir a alta qualidade e a personalização dos produtos, proporcionando uma experiência visualmente atraente e fácil de navegar para os clientes.

Calculadora

Desenvolvi a Calculadora Interativa utilizando HTML, CSS e JavaScript. Este projeto demonstra minhas habilidades em criação de interfaces e funcionalidades interativas, permitindo que os usuários realizem cálculos de forma prática e intuitiva. A calculadora foi projetada para ser responsiva e fácil de usar, oferecendo uma experiência de usuário agradável e eficiente para diferentes dispositivos e tamanhos de tela.

Login Playstation

Recriei a tela de login do PlayStation utilizando HTML, CSS e JavaScript. Este projeto é um exercício de design e desenvolvimento, focado em replicar com precisão a interface e a funcionalidade da tela de login original. A recriação demonstra minhas habilidades em front-end, incluindo a construção de interfaces responsivas e a implementação de interações intuitivas que proporcionam uma experiência de usuário semelhante à do ambiente PlayStation.
